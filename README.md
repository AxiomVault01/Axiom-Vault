# AXIOM-VAULT - An AI-driven internal audit and fraud detection platform designed for large Nigerian B2B organizations and government bodies.

AxiomVault is an artificial intelligence continuously monitors employee records and bank accounts to identify suspicious activities, duplicate accounts, and financial irregularities.

With AxiomVault, we deliver a usable, secure, and auditable internal fraud detection tool that enables internal auditors to identify high-risk payroll anomalies and insider collusion early. It is designed to prove value rapidly by uncovering suspicious payroll activity, supporting investigations with clear evidence, and converting potential losses into verifiable savings. Success at the MVP stage should justify further expansion into broader financial fraud coverage.


## Target Users

The is designed primarily for non-technical internal audit and compliance professionals within large organizations. Secondary users include fraud risk managers, compliance officers, and executive stakeholders who require high-level visibility into fraud exposure and financial risk. It's built on:

- React 19
- TypeScript
- Tailwind CSS v4


## Installation

### Prerequisites

To get started with Axiom-Vault, ensure you have the following prerequisites installed and set up:

- Node.js 18.x or later (recommended to use Node.js 20.x or later)

### Cloning the Repository

Clone the repository using the following command:

```bash
git clone https://github.com/AxiomVault01/Axiom-Vault.git
```

> Windows Users: place the repository near the root of your drive if you face issues while cloning.

1. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   ```

2. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```
